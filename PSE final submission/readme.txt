Room Booking System (Command Line Program)

The file room.txt is the data file consisting of details of all rooms, including room name, capacity, equipment and available timeslots

The script functions.py consists of all the functions and class to be run the the application program (app.py)

The script app.py is the main program, which is responsible for running the booking system, including functionality such as allowing users to 
view specifications, book rooms and lookup rooms based on certain criteria, such as capacity, timeslot and equipment.
